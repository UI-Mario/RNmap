## 主要功能

- 高德地图api，包括地图展示、定位、标识等
- 微信api，分享功能
- 二维码扫描，仅用于页面跳转
- 音视频播放
- 搜索
- AR，目前仅以AR.js方式实现
- 3d模型展示
